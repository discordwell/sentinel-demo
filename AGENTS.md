# Sentinel Demo

Static landing page for Sentinel — plain HTML/CSS/JS, no build step.

## Stack
- HTML5 / CSS3 / Vanilla JS
- Served via `python3 -m http.server 8050` for local dev
- Deployed via `deploy` skill

## Key Files
- `index.html` — single-page layout, 11 sections
- `style.css` — CSS custom properties, responsive breakpoints at 1024px/768px
- `grid-effect.js` — interactive hero grid hover effect
- `assets/logos/` — press logo SVGs
- `assets/team/` — team placeholder images

## Design Tokens
- Orange: #E8753A
- Cream: #F5F0E8
- Dark maroon: #7A2E2E
- Sage green: #8B9E8B
- Dark: #1A1A1A

## Fonts
- DM Mono (monospace headings/labels)
- DM Serif Display (large headings)
- System sans-serif (body)
